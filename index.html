Hi, I'm Oliver.

I'm a self-taught front-end developer based in London.

Passionate about creating user-friendly and stylish web projects, I am also a self-taught on the design end.

For web apps and websites my bread and butter is Javascript, with some React experience.

For prototyping and designs, I use Figma.

I started my journey off by learning HTML, CSS, and Javascript.

Once comfortable and confident with these I directed my attention to the React Library.

To improve my designing and creativity I started learning and using the design tool Figma, which I now use to start most projects.

I consider myself to be an excellent communicator; which helps me produce exactly what the client wants as well as giving them a clear understanding of what can be achieved.

I am also very critical and analytical, spotting errors and fine tuning code and UI/UX for the best aesthetic and practical output on projects.

All websites I make are designed to function on both desktop and mobile screen sizes.
